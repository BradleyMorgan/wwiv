WWIV BBS
=========

WWIV is compiled with the following compilers:
  
- MS Visual C++ 2015 Community Edition.
- GCC 4.9 on Linux (Debian or Ubuntu preferred, but not required)

You will need CMake 2.8 or later to build WWIV.

***
